# 🤖 InmoAsistente IA - MVP

Un asistente inmobiliario virtual inteligente diseñado para automatizar la atención a interesados en alquileres, calificar perfiles de estudiantes y derivar leads listos para cerrar a WhatsApp.

## 🚀 Características
- **Backend:** Desarrollado con FastAPI (Python) y base de datos relacional SQLite.
- **Inteligencia Artificial:** Integración fluida con modelos Llama 3.1 a través de la API de Groq (procesamiento ultra rápido y coste cero).
- **Frontend Interactivo:** Interfaz responsiva mobile-first (estilo smartphone) con ventana modal para validación de requisitos.
- **Flujo de Conversión:** Filtro inteligente de leads y redirección directa al canal de ventas de la inmobiliaria en WhatsApp.git branch -M main
